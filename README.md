# maxvw.xyz

"A digital sandbox where chaos meets code. No roadmaps, no promises, just raw exploration in the browser."

Welcome to the source code of **maxvw.xyz**. If you're looking for clean, documented, and production-ready code, you've probably taken a wrong turn at the internet. This is my personal playground for typographic experiments, questionable JavaScript decisions, and breaking things for the sake of "aesthetic."

## **🎨 Design Language**

* **Typography:** Brutalist, oversized, and uncomfortably bold.  
* **Color Palette:** Lake Blue (\#007187), Burnt Sienna (\#871600), and absolute white/black.  
* **Philosophy:** If the interface is too easy to understand, it's not experimental enough.

## **🛠 Tech Stack**

* **Framework:** React 18 (via CDN).  
* **Styling:** Tailwind CSS.  
* **Icons:** Lucide Icons.  
* **Hosting:** GitHub Pages.  
* **Logic:** Mostly Math.random() and sheer willpower.

## **📱 Mobile Support**

Wait, you're viewing this on a phone? My custom cursor already committed suicide to save your user experience. I've implemented a touch-aware logic that hides the oversized cursor on mobile devices, because I care about your screen real estate (sometimes).

## **🚀 Deployment**

This site is built into a single index.html file.

1. Push index.html to the main branch.  
2. Ensure the CNAME file contains maxvw.xyz.  
3. Wait for GitHub Actions to deploy.  
4. If it breaks, it's probably a DNS issue (it's always DNS).
